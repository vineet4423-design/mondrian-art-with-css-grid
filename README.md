# Mondrian Painting with CSS Grid

This project recreates a Piet Mondrian-style painting using pure HTML and CSS Grid.

## What I learned
- CSS Grid fundamentals
- grid-template-rows and grid-template-columns
- Using grid gaps to simulate Mondrian black lines
- Debugging pixel-perfect layout issues

## Tech used
- HTML
- CSS (Grid)
